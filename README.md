# Web Stylebook

[![English](https://img.shields.io/badge/lang-English-blue)](./README.en.md) [![한국어](https://img.shields.io/badge/lang-한국어-red)](./README.ko.md) [![日本語](https://img.shields.io/badge/lang-日本語-green)](./README.ja.md)

**Tired of every AI-generated site looking the same?**

An open-source web design stylebook for vibe coding: **33 style templates**, **9 fusion combinations**, and practical prompt workflow tools for building less generic AI-generated frontends.

**[Live Demo](https://webstylebook.com)** · **[Prompt Workflow](https://webstylebook.com/pages/prompt-workflow.html)** · **[Compare Styles](https://webstylebook.com/pages/compare.html)**

---

## What's Inside

- **33 Style Templates** — Full-page visual references with distinct typography, color, layout, and interaction direction
- **9 Fusion Combos** — Hybrid directions that blend two styles into one usable prompt target
- **4 Workflow Utilities** — Style comparison, color systems, prompt tips, and a step-by-step prompt generator
- **47 Sitemap URLs** — Static pages ready for search indexing on the canonical domain

| Category | Styles |
|----------|--------|
| Dark & Neon | Neon Drift, Cyberpunk Glitch, Midnight Noir, Holographic Fluid |
| Minimal & Editorial | Editorial Silence, Swiss Poster, Mono Type, Zen Minimalism, Quiet Utility |
| Organic & Warm | Earth Atelier, Soft Pastel, Aurora Gradient, Grain Texture, Organic Blob |
| Bold & Experimental | Kinetic Pop, Brutalist Grid, Y2K Retro, Liquid Metal, Duotone Bold |
| Modern & Clean | Glass Orbit, Bento Bloom, Mesh Gradient, Neumorphism, Notion Style |
| Tech & Product | Terminal Core, Console Launch, Platform Core, Runtime Signal, Framer Motion |
| Tactile & Print | Claymorphism, Retro Pixel, Risograph Print, Paper Cut |

## Use It With AI Coding Tools

1. Pick a visual direction from the gallery
2. Open the matching page or compare multiple styles side by side
3. Use the Prompt Workflow page to turn the style into implementation instructions
4. Paste the generated direction into Cursor, Claude, v0, Bolt, or your coding agent

## Quick Start

```bash
open index.html
# or
python3 -m http.server 4173
```

No framework, no build step. It is a static HTML/CSS/JS site deployed on Cloudflare Pages.
