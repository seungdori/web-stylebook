AI Request Prompt

Design a premium gallery/luxury brand landing page combining earthy, neutral nature tones with extreme whitespace and minimalist aesthetics.

[Conditions]
1. Typography: English titles in large serif (Cormorant Garamond), body text in the thinnest sans-serif font for contrast.
2. Component spacing: Section gaps default to 15vh or more, and padding inside components should be 2-3x that of typical UI.
3. Minimize visual stimulation: No flashy gradients, strong shadows, or border lines. Rhythm is created solely through font size and color (#3b3631, #8b7d6b) brightness differences.

Responsive:
- Mobile (~768px): Section gaps reduced to 8vh, serif title scaled down, single column
- Desktop: max-width 960px, generous 15vh+ section spacing maintained

Restrictions:
- No saturated or bright colors — strictly earth tones (#3b3631, #8b7d6b, #f5f0eb range)
- No box-shadows or border lines — whitespace is the only separator
- No animations or transitions — static elegance only

Output:
1) Typography scale (serif/sans-serif pairing, size, weight)
2) Spacing system and section rhythm
3) HTML/CSS code with responsive support

반응형:
- 모바일(~768px): 섹션 간격 8vh로 축소, 세리프 제목 크기 조정, 1열 레이아웃

- box-shadowやボーダーライン禁止 — 余白が唯一の区分
