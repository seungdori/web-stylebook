A lifestyle page crafted with natural textures and quiet contrast

자연 질감과 조용한 대비로 완성하는 라이프스타일 페이지

自然な質感と静かなコントラストで 仕上げるライフスタイルページ

This style creates an emotive yet readable interface using warm neutral backgrounds, low-brightness text, and restrained lines. Including "textured backgrounds," "low-saturation earthy accent points," and "soft card curvature" in your AI prompt produces stable results.

따뜻한 뉴트럴 배경, 저명도 텍스트, 절제된 라인으로 감성적이지만 읽기 쉬운 인터페이스를 만듭니다. AI 프롬프트에는 "질감이 느껴지는 배경", "낮은 채도의 흙빛 포인트", "부드러운 카드 곡률"을 포함하면 결과가 안정적입니다.

温かみのあるニュートラルな背景、低明度のテキスト、節制されたラインで感性的でありながら読みやすいインターフェースを作ります。AIプロンプトには「質感のある背景」「低彩度のアースカラーアクセント」「柔らかなカードの曲率」を含めると安定した結果が得られます。

Earth and plant-toned colors instead of bright neon
강한 네온 대신 토양·식물 계열 색상

텍스트 길이를 받아주는 넓은 카드

AI Request Prompt

Design a landing page in Earth Atelier style — natural lifestyle aesthetic with warm textures and quiet contrast.

COLOR TOKENS:
--sand: #efe6d6
--clay: #ab6b49
--leaf: #5f6a47
--ink: #2e2a24
--paper: #faf6ee
--card-bg: rgba(255, 255, 255, 0.42)
--card-border: #ccb596
--hero-border: #d9c9b0
--prompt-bg: #f5ecdd
--btn-border: #8e6a4c
--btn-text: #4d3c2f
--btn-bg: #f8f2e7
Background: radial-gradient(circle at 84% 12%, rgba(171, 107, 73, 0.18), transparent 30%),
  radial-gradient(circle at 12% 78%, rgba(95, 106, 71, 0.18), transparent 30%),
  var(--paper).
No other colors.

TYPOGRAPHY:
Heading: 'Gowun Batang' 700 (serif)
Body: 'Outfit' 400 / 500 / 600 / 700
Scale: clamp(1.9rem, 5vw, 4.1rem) for h1
Heading line-height: 1.2
Body line-height: 1.75
Lead paragraph max-width: 700px

UI:
Hero panel: border-radius 34px, background linear-gradient(145deg, #f3eadb 0%, #efe4d2 55%, #f7f1e6 100%), border 1px solid #d9c9b0, padding clamp(24px, 5vw, 48px), box-shadow inset 0 0 0 1px rgba(255, 255, 255, 0.8) + 0 18px 28px rgba(88, 67, 52, 0.13)
Note cards: 1px solid #ccb596, border-radius 16px, padding 14px, background rgba(255, 255, 255, 0.42)
Buttons: 1px solid #8e6a4c, color #4d3c2f, background #f8f2e7, border-radius 999px, padding 8px 14px

LAYOUT:
Content max-width: min(1040px, 91vw)
Page padding: 28px 0 84px
Note grid: auto-fit, minmax(200px, 1fr), gap 12px
Hero: left-aligned text, gradient textured panel

MOTION:
Entrance: translateY(12px) → 0, opacity 0 → 1, 650ms ease, stagger 100ms per card
Hover: subtle lift (translateY -2px), transition 250ms ease
No scroll-triggered animations. Respect prefers-reduced-motion.

RESPONSIVE:
768px: Note cards stack to 1 column, hero padding clamps down
1040px: Full grid layout, centered container

FORBIDDEN:
- Horizontal scroll at any viewport
- Neon or high-saturation accent colors
- Abrupt motion or bounce/elastic effects
- Cold blue or purple tones
- Sharp corners on hero or cards — maintain organic curvature
- Heavy drop shadows — only soft inset and ambient shadows

OUTPUT:
1) Color + typography tokens as CSS custom properties
2) Component structure: Hero panel with gradient, Note cards (3), CTA button
3) Semantic HTML + CSS with responsive support
