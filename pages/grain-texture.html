Film Grain
Aesthetic

Combine film grain with muted tones to bring analog aesthetics into digital form. In AI prompts, specify "noise texture overlay", "muted warm palette", "editorial serif typography" to achieve results that are both vintage and modern.

AI Request Prompt

Design a web landing page with film grain texture aesthetics.

Style conditions:
- Background: Dark brown (#1a1714) base, SVG feTurbulence grain overlay
- Place 1-2 subtle ambient lights (radial-gradient)
- Colors: Terracotta #e07a5f, sage #81b29a, muted beige #a09484
- Typography: Titles in serif (DM Serif Display), body in sans-serif (DM Sans)
- UI: Low-contrast borders, rounded cards, generous whitespace
- Accent: Small circular indicators for visual rhythm
- Motion: Minimal sequential entrance animations

Responsive:
- Mobile (~768px): Single column, grain overlay opacity reduced, serif titles scaled down
- Desktop: max-width 1000px, 2-column feature cards with generous gap

Restrictions:
- No saturated or neon colors — earthy muted palette only
- No sharp corners — minimum 12px border-radius
- No heavy box-shadows — depth through color layering, not elevation

Output:
1) Grain overlay CSS code (SVG feTurbulence)
2) Color tokens and typography system
3) Hero/Feature/CTA section HTML/CSS with responsive support
필름 그레인 텍스처 감성의 웹 랜딩 페이지를 디자인해줘.

스타일 조건:
- 배경: 어두운 브라운(#1a1714) 기반, SVG feTurbulence 그레인 오버레이
- 은은한 앰비언트 라이트(radial-gradient) 1~2개 배치
- 컬러: 테라코타 #e07a5f, 세이지 #81b29a, 뮤트 베이지 #a09484
- 타이포: 제목은 세리프(DM Serif Display), 본문은 산세리프(DM Sans)
- UI: 낮은 대비 테두리, 둥근 카드, 넓은 여백
- 포인트: 작은 원형 인디케이터로 시각 리듬 부여
- 모션: 최소한의 순차 등장 애니메이션

반응형:
- 모바일(~768px): 1열 레이아웃, 그레인 오버레이 투명도 축소, 세리프 제목 크기 조정
- 데스크톱: max-width 1000px, 2열 피처 카드에 넉넉한 간격

금지사항:
- 채도 높은 색상이나 네온 컬러 금지 — 어스 뮤트 팔레트만 허용
- 날카로운 모서리 금지 — 최소 12px border-radius
- 무거운 box-shadow 금지 — 높이감 대신 컬러 레이어링으로 깊이 표현

출력:
1) 그레인 오버레이 CSS 코드 (SVG feTurbulence)
2) 컬러 토큰과 타이포 시스템
3) 반응형 대응이 포함된 Hero/Feature/CTA 섹션 HTML/CSS
フィルムグレインテクスチャ感性のウェブランディングページをデザインしてください。

スタイル条件:
- 背景: ダークブラウン(#1a1714)ベース、SVG feTurbulenceグレインオーバーレイ
- 控えめなアンビエントライト(radial-gradient)を1〜2個配置
- カラー: テラコッタ#e07a5f、セージ#81b29a、ミュートベージュ#a09484
- タイポグラフィ: タイトルはセリフ(DM Serif Display)、本文はサンセリフ(DM Sans)
- UI: 低コントラストボーダー、丸みのあるカード、広い余白
- ポイント: 小さな円形インジケーターで視覚リズムを付与
- モーション: 最小限の順次登場アニメーション

レスポンシブ:
- モバイル(~768px): 1列レイアウト、グレインオーバーレイ透明度縮小、セリフタイトルサイズ調整
- デスクトップ: max-width 1000px、2列フィーチャーカードにゆとりある間隔

禁止事項:
- 彩度の高い色やネオンカラー禁止 — アーシーミュートパレットのみ
- 鋭い角禁止 — 最小12px border-radius
- 重いbox-shadow禁止 — 高さではなくカラーレイヤリングで深みを表現

出力:
1) グレインオーバーレイCSSコード(SVG feTurbulence)
2) カラートークンとタイポグラフィシステム
3) レスポンシブ対応を含むHero/Feature/CTAセクションHTML/CSS
