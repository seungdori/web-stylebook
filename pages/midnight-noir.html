MIDNIGHT NOIR

A style that creates a premium mood by placing gold accents on a dark base. When prompting, instead of the word "luxury," specify the components clearly: "deep black background + champagne gold lines + serif headlines" for accurate results.

어두운 바탕에 골드 포인트를 얹어 프리미엄 무드를 만드는 스타일입니다. 요청할 때는 "럭셔리"라는 단어보다 "딥 블랙 배경 + 샴페인 골드 라인 + 세리프 헤드라인"처럼 구성 요소를 명확히 지정해야 결과가 정확합니다.

暗い背景にゴールドのアクセントを添えてプレミアムな雰囲気を作るスタイルです。リクエスト時には「ラグジュアリー」という言葉よりも「ディープブラック背景＋シャンパンゴールドライン＋セリフ見出し」のように構成要素を明確に指定すると正確な結果が得られます。

Low-light contrast instead of high contrast
고대비 대신 저조도 대비
高コントラストの代わりに低照度コントラスト
Serif titles + restrained body text
세리프 타이틀 + 절제된 본문
セリフタイトル＋節制された本文
Glossy effects limited to accent areas only
광택감은 강조 영역에만 제한
光沢感はアクセント領域のみに制限

AI Request Prompt

Design a landing page in Midnight Noir style — champagne gold accents on pitch-black, cinematic depth.

COLOR TOKENS:
--bg: #0f0f12
--bg-gradient-start: #0a0a0c
--panel: rgba(18, 18, 21, 0.72)
--border: #3e372a
--text: #eee6d7
--accent-1: #d6b577
Background bleed: radial-gradient(circle at 80% 12%, rgba(214,181,119,0.22), transparent 34%), radial-gradient(circle at 10% 90%, rgba(214,181,119,0.12), transparent 34%).
No other colors.

TYPOGRAPHY:
Heading: "Cormorant Garamond" 700, tracking 0.02em
Body: "Syne" 400
Scale: clamp(2.2rem, 6vw, 5rem) for h1
Body line-height: 1.75
Heading line-height: 0.9

UI:
Hero panel: border-radius 28px, border 1px solid var(--border), bg linear-gradient(140deg, rgba(30,29,26,0.9), rgba(15,15,18,0.94)), box-shadow 0 24px 46px rgba(0,0,0,0.55).
Item cards: border-radius 14px, border 1px solid var(--border), bg rgba(18,18,21,0.72), padding 12px.
Prompt box: border-radius 18px, border 1px solid var(--border), bg rgba(12,12,15,0.84).
Buttons: pill shape (border-radius 999px), border 1px solid var(--accent-1), color var(--accent-1), bg transparent.

LAYOUT:
Content max-width: min(1040px, 92vw)
Page padding: 30px 0 80px
Hero padding: clamp(24px, 5vw, 52px)
Card grid: repeat(auto-fit, minmax(180px, 1fr)), gap 10px
Lead max-width: 700px

MOTION:
Entrance: translateY(12px) → 0, opacity 0 → 1, 700ms ease, stagger 100ms per item
Hover: none specified — keep restrained
Respect prefers-reduced-motion.

RESPONSIVE:
768px: card grid collapses to 1 column, hero padding reduces via clamp
1024px: full 3-column auto-fit grid, max-width 1040px layout

FORBIDDEN:
- High-saturation or neon colors
- White or light backgrounds
- Glow effects or excessive decoration
- Sans-serif headings (serif only for h1/h2)
- Horizontal scroll at any viewport
- More than one accent color

OUTPUT:
1) Color + typography tokens as CSS custom properties
2) Hero / Story / Proof (3-column items) / CTA structure
3) Semantic HTML + CSS with responsive support
