Restrictions:
- No opaque solid backgrounds on cards — always translucent glass
- No flat colors — all surfaces must show background bleed-through
- No warm tones — stay within cool magenta/aqua/purple spectrum

Output:
1) Holographic orb animation CSS
2) Glass panel component system (blur, border, highlight)
3) HTML/CSS code with responsive support
