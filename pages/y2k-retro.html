AI Request Prompt

Create a web page with a Y2K Windows 95 (or Web 1.0) theme inspired by late 1990s to early 2000s internet aesthetics.

Requirements:
1) Borders: All components (windows, buttons) use 1-2px solid angular borders, no rounded corners.
2) Bevel/Emboss: border-top/left is #ffffff (light side), border-bottom/right is #000000 (dark side). Additional inset box-shadow for internal depth.
3) Window structure: Title bar (linear-gradient 90deg, #000080 to #1084d0) + minimize/maximize/close buttons + white inner body.
4) Background: Teal (#008080) pattern background, area outside windows feels like a desktop.
5) Typography: Impact for headings, MS Sans Serif/Comic Sans for body, fluorescent accents (Magenta #ff00ff, Cyan #00ffff).
6) Marquee: CSS animation (translateX 100% to -100%, 10s linear infinite) for scrolling text.
7) Buttons: Bevel borders, active state inverts light/dark sides for pressed effect.

Responsive:
- Mobile: Window width 92vw, reduced font sizes
- Desktop: max-width 1000px, window shadow (4px 4px 0 rgba(0,0,0,0.5))

Restrictions:
- No rounded corners (border-radius) allowed
- No modern gradients or glassmorphism effects
- No sleek sans-serif fonts

Output:
1) Window wrapper (title bar/body/buttons) structure and bevel CSS variables
2) Hero window / marquee / prompt window layout
3) HTML/CSS code with responsive support

1990年代後半から2000年代初頭のインターネット感性のY2K Windows 95（またはWeb 1.0）テーマでウェブページを作成してください。

要件:
1) 外枠: すべてのコンポーネント（ウィンドウ、ボタン）に1〜2pxソリッドの角張った外枠、角丸禁止。
2) ベベル/エンボス: border-top/leftは#ffffff（明るい面）、border-bottom/rightは#000000（暗い面）。追加でinset box-shadowで内部の立体感。
3) ウィンドウ構造: タイトルバー（linear-gradient 90deg, #000080→#1084d0）+ 最小化/最大化/閉じるボタン + 白い内部body。
4) 背景: ティール（#008080）パターン背景、ウィンドウ外の領域はデスクトップ風。
5) タイポグラフィ: 英文はImpact、本文はMS Sans Serif/Comic Sans、蛍光色（Magenta #ff00ff、Cyan #00ffff）アクセント。
6) マーキー: CSSアニメーション（translateX 100%→-100%、10s linear infinite）でテキストスクロールを実装。
7) ボタン: ベベルボーダー、active時に明/暗面反転で押下効果。

レスポンシブ:
- モバイル: ウィンドウ幅92vw、フォントサイズ縮小
- デスクトップ: max-width 1000px、ウィンドウシャドウ（4px 4px 0 rgba(0,0,0,0.5)）

禁止事項:
- 角丸（border-radius）の使用禁止
- モダンなグラデーションやグラスモーフィズム効果禁止
- 洗練されたサンセリフフォントの使用禁止

出力:
1) ウィンドウラッパー（タイトルバー/body/ボタン）構造とベベルCSS変数
2) ヒーローウィンドウ / マーキー / プロンプトウィンドウレイアウト
3) レスポンシブ対応を含むHTML/CSSコード
