Interface

AI Request Prompt

Design an ultra-soft pastel aesthetic web landing page.

Style conditions:
- Background: Warm cream (#fef7f0), pastel ambient radial-gradient
- Colors: Peach #ff9a76, sky blue #a8d8ea, lavender #c3b1e1, mint #8ecf92
- UI: 999px border-radius pill shapes, translucent cards, subtle shadows
- Typography: Nunito 800 (titles), 400 (body), warm brown text
- Elements: Color dot indicators, badges, pill-shaped tags
- Buttons: Solid fill with shadow, lift effect on hover
- Motion: Scale pop animation, sequential appearance

Responsive:
- Mobile (~768px): Single column stack, pill sizes scaled down, animations simplified
- Desktop: max-width 1000px, 2-3 column feature grid

Restrictions:
- No dark backgrounds or dark text — warm light palette only
- No sharp corners — all elements must be fully rounded (pill or circle)
- No saturated/neon colors — keep all tones pastel and soft

Output:
1) Pastel palette and color tokens
2) Pill-shaped UI component system
3) Hero/Feature/CTA section HTML/CSS with responsive support
울트라 소프트 파스텔 감성의 웹 랜딩 페이지를 디자인해줘.

스타일 조건:
- 배경: 따뜻한 크림(#fef7f0), 파스텔 색상 앰비언트 radial-gradient
- 컬러: 피치 #ff9a76, 스카이블루 #a8d8ea, 라벤더 #c3b1e1, 민트 #8ecf92
- UI: 999px border-radius 필 모양, 반투명 카드, 미세한 그림자
- 타이포: Nunito 800 (제목), 400 (본문), 따뜻한 브라운 텍스트
- 요소: 컬러 도트 인디케이터, 배지, 필 형태 태그
- 버튼: 그림자가 있는 솔리드 필, 호버 시 lift 효과
- 모션: scale pop 애니메이션, 순차 등장

반응형:
- 모바일(~768px): 1열 스택, 필 사이즈 축소, 애니메이션 단순화
- 데스크톱: max-width 1000px, 2~3열 피처 그리드

금지사항:
- 다크 배경이나 어두운 텍스트 금지 — 따뜻한 밝은 팔레트만 사용
- 날카로운 모서리 금지 — 모든 요소 완전 라운드(필 또는 원형)
- 채도 높은/네온 컬러 금지 — 모든 톤 파스텔과 소프트 유지

출력:
1) 파스텔 팔레트와 컬러 토큰
2) 필 형태 UI 컴포넌트 시스템
3) 반응형 대응이 포함된 Hero/Feature/CTA 섹션 HTML/CSS
ウルトラソフトパステル感性のウェブランディングページをデザインしてください。

スタイル条件:
- 背景: 温かみのあるクリーム(#fef7f0)、パステルカラーのアンビエントradial-gradient
- カラー: ピーチ#ff9a76、スカイブルー#a8d8ea、ラベンダー#c3b1e1、ミント#8ecf92
- UI: 999px border-radiusのピル型、半透明カード、微細なシャドウ
- タイポグラフィ: Nunito 800(タイトル)、400(本文)、温かみのあるブラウンテキスト
- 要素: カラードットインジケーター、バッジ、ピル型タグ
- ボタン: シャドウ付きソリッドフィル、ホバー時lift効果
- モーション: scale popアニメーション、順次登場

レスポンシブ:
- モバイル(~768px): 1列スタック、ピルサイズ縮小、アニメーション簡略化
- デスクトップ: max-width 1000px、2〜3列フィーチャーグリッド

禁止事項:
- ダーク背景や暗いテキスト禁止 — 温かみのある明るいパレットのみ
- 鋭い角禁止 — 全要素完全ラウンド(ピルまたは円形)
- 彩度の高い/ネオンカラー禁止 — 全トーンパステルとソフトを維持

出力:
1) パステルパレットとカラートークン
2) ピル型UIコンポーネントシステム
3) レスポンシブ対応を含むHero/Feature/CTAセクションHTML/CSS
