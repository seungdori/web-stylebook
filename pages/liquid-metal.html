AI Request Prompt

Design a liquid metal (chrome) aesthetic web landing page.

Style conditions:
- Background: Subtle light reflection gradient on pure black (#0c0c0c)
- Text: Chrome effect — multi-step grayscale linear-gradient + background-clip:text
- UI: 1px highlight at top of cards, subtle internal gradients
- Colors: Monochrome silver palette (#ffffff to #6b6b6b), rely on texture without color
- Motion: Diagonal light sweep animation, chrome text shift effect
- Layout: Generous whitespace, large titles, restrained element count

Responsive:
- Mobile (~768px): Single column, chrome text gradient simplified, sweep animation disabled
- Desktop: max-width 1000px, 2-column feature layout

Restrictions:
- No color accents — monochrome silver only
- No flat matte surfaces — every panel needs subtle gradient or inset highlight
- No border-radius above 20px — keep edges sharp and premium

Output:
1) Chrome text gradient CSS and color tokens
2) Light sweep animation implementation
3) Hero/Feature section HTML/CSS with responsive support
리퀴드 메탈(크롬) 감성의 웹 랜딩 페이지를 디자인해줘.

스타일 조건:
- 배경: 순수 블랙(#0c0c0c) 위 미세한 빛 반사 그라데이션
- 텍스트: 크롬 효과 — 다단 그레이스케일 linear-gradient + background-clip:text
- UI: 카드에 상단 1px 하이라이트, 미세한 내부 그라데이션
- 컬러: 모노크롬 실버 팔레트 (#ffffff ~ #6b6b6b), 컬러 없이 질감으로 승부
- 모션: 사선 빛 sweep 애니메이션, 크롬 텍스트 shift 효과
- 레이아웃: 넉넉한 여백, 큰 제목, 절제된 요소 수

반응형:
- 모바일(~768px): 1열 레이아웃, 크롬 텍스트 그라디언트 단순화, sweep 애니메이션 비활성
- 데스크톱: max-width 1000px, 2열 피처 레이아웃

금지사항:
- 컬러 포인트 금지 — 모노크롬 실버만 허용
- 평면 매트 표면 금지 — 모든 패널에 미세 그라디언트 또는 인셋 하이라이트 필수
- 20px 초과 border-radius 금지 — 날카롭고 프리미엄한 엣지 유지

출력:
1) 크롬 텍스트 그라데이션 CSS 및 컬러 토큰
2) 빛 sweep 애니메이션 구현
3) 반응형 대응이 포함된 Hero/Feature 섹션 HTML/CSS
リキッドメタル(クローム)感性のウェブランディングページをデザインしてください。

スタイル条件:
- 背景: 純粋なブラック(#0c0c0c)の上に微細な光反射グラデーション
- テキスト: クローム効果 — 多段グレースケールlinear-gradient + background-clip:text
- UI: カード上部に1pxハイライト、微細な内部グラデーション
- カラー: モノクロームシルバーパレット(#ffffff〜#6b6b6b)、色なしで質感で勝負
- モーション: 斜めの光sweepアニメーション、クロームテキストshift効果
- レイアウト: ゆったりとした余白、大きなタイトル、抑制された要素数

レスポンシブ:
- モバイル(~768px): 1列レイアウト、クロームテキストグラデーション簡略化、sweepアニメーション無効
- デスクトップ: max-width 1000px、2列フィーチャーレイアウト

禁止事項:
- カラーアクセント禁止 — モノクロームシルバーのみ
- フラットマット表面禁止 — 全パネルに微細グラデーションまたはインセットハイライト必須
- 20px超のborder-radius禁止 — シャープでプレミアムなエッジを維持

出力:
1) クロームテキストグラデーションCSSとカラートークン
2) 光sweepアニメーション実装
3) レスポンシブ対応を含むHero/FeatureセクションHTML/CSS
