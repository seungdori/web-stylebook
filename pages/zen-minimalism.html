The value of Silence

A zen minimalism style that uses extreme whitespace and elegant serif fonts to focus attention solely on the essence of the message.

극단적인 여백과 우아한 세리프 폰트를 사용하여, 전달하고자 하는 본질에만 집중하게 만드는 젠 미니멀리즘 스타일.

極端な余白と優雅なセリフフォントを使用して、伝えたい本質にのみ集中させるゼンミニマリズムスタイルです。

Design Parameters

Design an exhibition webpage with extreme whitespace and minimalist aesthetics that retains only the essence.

[Design Essence]
1. Layout: Use 70% of the screen as whitespace. max-width 800px, padding 80px 40px.
2. Typography: English titles in Cormorant Garamond (weight 300, italic usage), body in Noto Sans KR (weight 300). Title clamp(3rem, 6vw, 5.5rem), body line-height 1.9.
3. Color: Warm off-white (#faf9f7) background, soft charcoal (#2c2c2a) text, muted (#555/#999) hierarchy.
4. Dividers: Thin vertical lines (1px #ddd, height 100px) for section separation, margin 100px auto.
5. Buttons: 1px line style, hover fills background (#2c2c2a) + text inversion (#fff), 0.3s transition.
6. Motion: Full 2s ease-in fade in, leisurely with no quick transitions.

[Responsive]
- Mobile: Padding 40px 20px, auto-shrink title size (clamp)
- Desktop: max-width 800px centered

[Restrictions]
- No intense colors or neon effects
- No complex grids or multi-column layouts
- No fast motion or bounce/spring effects
- No icons/illustrations/images

[Output]
1) Typography scale (h1/h2/body/manifesto) and color tokens
2) Header / Title / Manifesto / Divider / Content / CTA structure
3) HTML/CSS code with whitespace system and fade animations
오직 본질만을 남기고 극대화된 여백과 미니멀리즘 미학을 갖춘 전시 웹페이지를 설계해줘.

[디자인 에센스]
1. 레이아웃: 화면의 70%를 여백(Whitespace)으로 활용. max-width 800px, 패딩 80px 40px.
2. 타이포: 영문 타이틀은 Cormorant Garamond(weight 300, italic 활용), 본문은 Noto Sans KR(weight 300). 제목 clamp(3rem, 6vw, 5.5rem), 본문 줄간격 1.9.
3. 컬러: 따뜻한 미색(#faf9f7) 배경, 부드러운 차콜(#2c2c2a) 텍스트, 뮤트(#555/#999) 계층.
4. 구분선: 가늘고 긴 세로선(1px #ddd, 높이 100px)으로 섹션 분리, margin 100px auto.
5. 버튼: 1px 라인 스타일, hover 시 배경 채움(#2c2c2a) + 텍스트 반전(#fff), 0.3s 전환.
6. 모션: 전체 2s ease-in 페이드 인, 빠른 전환 없이 여유롭게.

[반응형]
- 모바일: 패딩 40px 20px, 제목 크기 자동 축소(clamp)
- 데스크톱: max-width 800px 중앙 정렬

[금지사항]
- 강렬한 색상이나 네온 효과 사용 금지
- 복잡한 그리드나 다단 레이아웃 금지
- 빠른 모션이나 bounce/spring 효과 금지
- 아이콘/일러스트/이미지 사용 금지

[출력]
1) 타이포 스케일(h1/h2/body/manifesto)과 컬러 토큰
2) Header / Title / Manifesto / Divider / Content / CTA 구조
3) 여백 시스템과 페이드 애니메이션 포함 HTML/CSS 코드
本質のみを残し、極大化された余白とミニマリズム美学を持つ展示ウェブページを設計してください。

[デザインエッセンス]
1. レイアウト: 画面の70%をホワイトスペースとして活用。max-width 800px、パディング80px 40px。
2. タイポグラフィ: 英文タイトルはCormorant Garamond(weight 300、italic活用)、本文はNoto Sans KR(weight 300)。タイトルclamp(3rem, 6vw, 5.5rem)、本文行間1.9。
3. カラー: 温かみのあるオフホワイト(#faf9f7)背景、柔らかいチャコール(#2c2c2a)テキスト、ミュート(#555/#999)階層。
4. 区切り線: 細く長い縦線(1px #ddd、高さ100px)でセクション分離、margin 100px auto。
5. ボタン: 1pxラインスタイル、hover時背景塗りつぶし(#2c2c2a) + テキスト反転(#fff)、0.3sトランジション。
6. モーション: 全体2s ease-inフェードイン、素早い遷移なくゆったりと。

[レスポンシブ]
- モバイル: パディング40px 20px、タイトルサイズ自動縮小(clamp)
- デスクトップ: max-width 800px中央揃え

[禁止事項]
- 強烈な色やネオン効果の使用禁止
- 複雑なグリッドや多段レイアウト禁止
- 素早いモーションやbounce/spring効果禁止
- アイコン/イラスト/画像の使用禁止

[出力]
1) タイポスケール(h1/h2/body/manifesto)とカラートークン
2) Header / Title / Manifesto / Divider / Content / CTA構造
3) 余白システムとフェードアニメーションを含むHTML/CSSコード
