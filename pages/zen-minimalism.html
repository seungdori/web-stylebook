RETURN TO HUB

The value of Silence

A zen minimalism style that uses extreme whitespace and elegant serif fonts to focus attention solely on the essence of the message.

극단적인 여백과 우아한 세리프 폰트를 사용하여, 전달하고자 하는 본질에만 집중하게 만드는 젠 미니멀리즘 스타일.

極端な余白と優雅なセリフフォントを使用して、伝えたい本質にのみ集中させるゼンミニマリズムスタイルです。

Design Parameters

Design a landing page in Zen Minimalism style — contemplative whitespace with restrained typographic elegance.

COLOR TOKENS:
--bg: #faf9f7
--text: #2c2c2a
--text-muted: #555555
--text-dim: #999999
--link: #888888
--link-hover: #2c2c2a
--divider: #dddddd
--prompt-bg: #ffffff
--prompt-border: #eeeeee
--prompt-text: #444444
No other colors.

TYPOGRAPHY:
Heading: Cormorant Garamond 300, italic for emphasis
Section title: Cormorant Garamond 400, 2rem
Body: Noto Sans 300
Button: Noto Sans 400
Scale: 0.85rem / 0.95rem / 1.3rem / 2rem / clamp(3rem, 6vw, 5.5rem)
Heading line-height: 1.1
Body line-height: 1.8 (pre), 1.9 (manifesto)
-webkit-font-smoothing: antialiased on body

UI:
- Prompt box: background #fff, border 1px solid #eee, padding 40px, box-shadow 0 20px 40px rgba(0,0,0,0.02)
- Section titles: border-bottom 1px solid #ddd, padding-bottom 20px, margin-bottom 40px
- Vertical divider: 1px wide, 100px tall, background #ddd, margin 0 auto 100px (centered)
- Button: background none, border 1px solid #2c2c2a, color #2c2c2a, padding 12px 30px, display block, margin 40px auto 0
- Button hover: background #2c2c2a, color #fff, transition all 0.3s
- Header links: color #888, font-size 0.85rem, letter-spacing 0.05em; hover color #2c2c2a, transition 0.3s

LAYOUT:
Container: max-width 800px, margin 0 auto, padding 80px 40px
Header: text-align center, margin-bottom 120px
Manifesto: font-size 1.3rem, text-align center, max-width 600px, margin 0 auto 100px, color #555
Title: text-align center, margin 0 0 40px
All content single-column, center-aligned. ~70% whitespace ratio.

MOTION:
Header entrance: @keyframes fade — from opacity 0 translateY(10px) to opacity 1 translateY(0)
Duration: 2s ease-in
Button hover: all 0.3s transition
Header link: color 0.3s transition
No scroll animations, no parallax, no bounce effects.

RESPONSIVE:
- Mobile: padding 40px 20px, title auto-scales via clamp, manifesto max-width 100%
- Desktop: max-width 800px centered, padding 80px 40px preserved

FORBIDDEN:
- Intense, saturated, or neon colors
- Multi-column grids or complex layouts
- Fast animations or bounce/spring easing functions
- Icons, illustrations, or photographic imagery
- Box-shadows stronger than rgba(0,0,0,0.02)

OUTPUT:
1) Full color token list and typography scale with font weights
2) Header / Title / Manifesto / Divider / Content / CTA vertical structure
3) Single-file HTML/CSS with fade keyframe and whitespace system

금지사항:
- 강렬하거나 채도 높은 네온 컬러

- 아이콘, 일러스트, 사진 이미지
- rgba(0,0,0,0.02) 이상의 box-shadow

출력:
1) font-weight 포함 전체 색상 토큰 목록과 타이포그래피 스케일
2) 헤더 / 타이틀 / 매니페스토 / 구분선 / 콘텐츠 / CTA 수직 구조
3) fade 키프레임과 여백 시스템 포함 단일 파일 HTML/CSS
