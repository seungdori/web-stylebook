The Analog Manifesto
아날로그 선언문
アナログ・マニフェスト
PROOF 교정 校正
STET 유지 ママ

We believe in the digital perfection of tactile imperfection of analog processes. Every grain particle is a flaw fingerprint, every keystroke leaves a physical impression on the page. The slight misalignment of characters, the uneven ink density across the ribbon's travel — these are not defects but evidence of the human hand at work.

우리는 아날로그 과정의 디지털 완벽함 촉각적 불완전함을 믿습니다. 모든 입자는 결함 지문이며, 모든 키 입력은 페이지에 물리적 인상을 남깁니다. 문자의 미세한 어긋남, 리본 이동에 따른 불균일한 잉크 농도 — 이것들은 결함이 아니라 인간의 손길의 증거입니다.

私たちはアナログプロセスのデジタルな完璧さ 触覚的な不完全さを信じています。すべての粒子は欠陥 指紋であり、すべてのキー入力はページに物理的な印象を残します。文字のわずかなずれ、リボンの移動による不均一なインク濃度 — これらは欠陥ではなく、人の手仕事の証です。

AI Request Prompt

Design a landing page in "Analog Press" fusion style — combining Grain Texture's film photography aesthetic with Mono Type's typewriter discipline.

COLOR TOKENS:
--bg: #1a1714 (warm darkroom dark)
--text: #e8ddd0 (warm photographic paper cream)
--accent: #e07a5f (terracotta / safelight red)
--accent-2: #81b29a (sage / chemical tint)
--muted: #8a7e6e (aged metal / worn key)
--line: rgba(232, 221, 208, 0.12)
--card-bg: rgba(38, 34, 28, 0.85)
No other colors.

TYPOGRAPHY:
- Headlines: 'DM Serif Display', serif — elegant, editorial, for film strip titles
- Body/UI: 'Space Mono', monospace — typewriter feel for all running text
- Headline size: clamp(2.2rem, 5.5vw, 4rem)
- Body: font-size 0.78–0.85rem, line-height: 1.65–2.0, letter-spacing: 0.08em

TEXTURE & GRAIN:
- SVG noise filter overlay on body::after with opacity 0.04
- Character grid on body::before using linear-gradient with 1ch/1.6em grid
- Film sprocket holes: small rounded rectangles in sidebar columns (14×10px)
- All borders use var(--line), no rounded corners (border-radius: 0)

LAYOUT ELEMENTS:
- Film negative strip header with sprocket holes on both sides, frame number, and serif title
- Contact sheet grid: 4-column grid with 2px gap, each frame has number, text sample, and exposure metadata
- Press proof manuscript: bordered section with serif title, monospace body text, red editor marks (strikethrough + caret inserts), PROOF stamp rotated -3deg
- Dual color palette: typewriter ribbon (square swatches) vs film canister (circular swatches)
- Exposure test strip: 5-column row showing same text at increasing font-weight and opacity (0.25 → 1.0)
- Vintage photo caption: film strip format with sprocket holes, timestamp, and monospace quote

DARKROOM EFFECTS:
- Safelight glow: radial-gradient rgba(224,122,95,0.06) on key sections
- Hover states: inset box-shadow with safelight color
- Editor marks: accent color for strikethrough, accent-2 for insertions with caret (^) above

MOTION:
- Minimal — only hover transitions (0.3s ease) on interactive elements
- No keyframe animations — stillness reflects analog patience

RESPONSIVE:
- Desktop (>720px): 4-column contact sheet, 5-column exposure strip, 2-column palette
- Tablet (<=720px): 2-column contact sheet, 3-column exposure strip, 1-column palette
- Mobile (<=480px): 1-column contact sheet, 2-column exposure strip

FORBIDDEN:
- Rounded corners (border-radius) on content elements — sharp edges only
- Bright saturated colors — muted analog palette only
- Sans-serif body text — all body text must be monospace
- Decorative illustrations — only typographic and photographic metaphors
- Smooth gradients as backgrounds — texture and grain are mandatory

OUTPUT:
1) Film strip + typewriter fusion layout system
2) SVG grain overlay + character grid + safelight glow effects
3) Full HTML/CSS with responsive breakpoints and darkroom color tokens
"Analog Press" 퓨전 스타일의 랜딩 페이지를 디자인해줘 — Grain Texture의 필름 사진 미학과 Mono Type의 타자기 규율을 결합한 스타일.

색상 토큰:
--bg: #1a1714 (따뜻한 암실 어둠)
--text: #e8ddd0 (따뜻한 인화지 크림)
--accent: #e07a5f (테라코타 / 세이프라이트 레드)
--accent-2: #81b29a (세이지 / 화학 틴트)
--muted: #8a7e6e (낡은 금속 / 마모된 키)
--line: rgba(232, 221, 208, 0.12)
--card-bg: rgba(38, 34, 28, 0.85)
이 외의 색상 사용 금지.

타이포그래피:
- 헤드라인: 'DM Serif Display', serif — 우아하고 에디토리얼, 필름 스트립 제목용
- 본문/UI: 'Space Mono', monospace — 모든 본문 텍스트에 타자기 느낌
- 헤드라인 크기: clamp(2.2rem, 5.5vw, 4rem)
- 본문: font-size 0.78–0.85rem, line-height: 1.65–2.0, letter-spacing: 0.08em

텍스처 & 그레인:
- body::after에 SVG 노이즈 필터 오버레이, opacity 0.04
- body::before에 linear-gradient로 1ch/1.6em 문자 그리드
- 필름 스프로킷 홀: 사이드바 열에 작은 둥근 직사각형 (14×10px)
- 모든 테두리는 var(--line) 사용, 둥근 모서리 없음 (border-radius: 0)

레이아웃 요소:
- 양쪽에 스프로킷 홀이 있는 필름 네거티브 스트립 헤더, 프레임 번호, 세리프 제목
- 컨택트 시트 그리드: 4열 그리드, 2px 갭, 각 프레임에 번호, 텍스트 샘플, 노출 메타데이터
- 교정쇄 원고: 테두리 있는 섹션에 세리프 제목, 모노스페이스 본문, 빨간 교정 표시 (취소선 + 캐럿 삽입), -3deg 회전된 PROOF 스탬프
- 이중 색상 팔레트: 타자기 리본 (정사각형 스와치) vs 필름 캐니스터 (원형 스와치)
- 노출 테스트 스트립: 같은 텍스트를 증가하는 font-weight와 opacity (0.25 → 1.0)로 보여주는 5열 행
- 빈티지 사진 캡션: 스프로킷 홀, 타임스탬프, 모노스페이스 인용문이 있는 필름 스트립 형식

암실 효과:
- 세이프라이트 글로우: 주요 섹션에 radial-gradient rgba(224,122,95,0.06)
- 호버 상태: 세이프라이트 컬러의 inset box-shadow
- 교정 표시: 취소선은 accent 컬러, 삽입은 위에 캐럿(^)과 함께 accent-2 컬러

모션:
- 최소 — 인터랙티브 요소에만 hover 전환 (0.3s ease)
- 키프레임 애니메이션 없음 — 고요함이 아날로그의 인내를 반영

반응형:
- 데스크톱 (>720px): 4열 컨택트 시트, 5열 노출 스트립, 2열 팔레트
- 태블릿 (<=720px): 2열 컨택트 시트, 3열 노출 스트립, 1열 팔레트
- 모바일 (<=480px): 1열 컨택트 시트, 2열 노출 스트립

금지사항:
- 콘텐츠 요소에 둥근 모서리 (border-radius) — 날카로운 모서리만
- 밝은 채도 높은 색상 — 음소거된 아날로그 팔레트만
- 산세리프 본문 텍스트 — 모든 본문은 반드시 모노스페이스
- 장식적 일러스트 — 타이포그래피와 사진 메타포만
- 부드러운 그라데이션 배경 — 텍스처와 그레인은 필수

출력:
1) 필름 스트립 + 타자기 퓨전 레이아웃 시스템
2) SVG 그레인 오버레이 + 문자 그리드 + 세이프라이트 글로우 효과
3) 반응형 브레이크포인트와 암실 색상 토큰을 포함한 전체 HTML/CSS
"Analog Press"フュージョンスタイルのランディングページをデザインしてください — Grain Textureのフィルム写真美学とMono Typeのタイプライター規律を融合したスタイル。

カラートークン:
--bg: #1a1714（温かい暗室の闇）
--text: #e8ddd0（温かい印画紙クリーム）
--accent: #e07a5f（テラコッタ / セーフライトレッド）
--accent-2: #81b29a（セージ / 化学ティント）
--muted: #8a7e6e（古びた金属 / 摩耗したキー）
--line: rgba(232, 221, 208, 0.12)
--card-bg: rgba(38, 34, 28, 0.85)
これ以外の色は使用禁止。

タイポグラフィ:
- 見出し: 'DM Serif Display', serif — エレガントでエディトリアル、フィルムストリップタイトル用
- 本文/UI: 'Space Mono', monospace — すべての本文テキストにタイプライター感
- 見出しサイズ: clamp(2.2rem, 5.5vw, 4rem)
- 本文: font-size 0.78–0.85rem, line-height: 1.65–2.0, letter-spacing: 0.08em

テクスチャ & グレイン:
- body::afterにSVGノイズフィルターオーバーレイ、opacity 0.04
- body::beforeにlinear-gradientで1ch/1.6emの文字グリッド
- フィルムスプロケットホール: サイドバー列の小さな角丸矩形（14×10px）
- すべてのボーダーはvar(--line)を使用、角丸なし（border-radius: 0）

レイアウト要素:
- 両側にスプロケットホールのあるフィルムネガストリップヘッダー、フレーム番号、セリフタイトル
- コンタクトシートグリッド: 4列グリッド、2pxギャップ、各フレームに番号、テキストサンプル、露出メタデータ
- 校正刷り原稿: ボーダー付きセクションにセリフタイトル、等幅本文、赤い校正マーク（取り消し線 + キャレット挿入）、-3deg回転のPROOFスタンプ
- 二重カラーパレット: タイプライターリボン（正方形スウォッチ） vs フィルムキャニスター（円形スウォッチ）
- 露出テストストリップ: 同じテキストをfont-weightとopacity（0.25→1.0）を増加させて表示する5列行
- ビンテージ写真キャプション: スプロケットホール、タイムスタンプ、等幅引用文のフィルムストリップ形式

暗室エフェクト:
- セーフライトグロー: 主要セクションにradial-gradient rgba(224,122,95,0.06)
- ホバー状態: セーフライトカラーのinset box-shadow
- 校正マーク: 取り消し線はアクセントカラー、挿入はキャレット（^）付きでaccent-2カラー

モーション:
- 最小 — インタラクティブ要素にのみhoverトランジション（0.3s ease）
- キーフレームアニメーションなし — 静けさがアナログの忍耐を反映

レスポンシブ:
- デスクトップ（>720px）: 4列コンタクトシート、5列露出ストリップ、2列パレット
- タブレット（<=720px）: 2列コンタクトシート、3列露出ストリップ、1列パレット
- モバイル（<=480px）: 1列コンタクトシート、2列露出ストリップ

禁止事項:
- コンテンツ要素の角丸（border-radius） — シャープなエッジのみ
- 明るい彩度の高い色 — ミュートされたアナログパレットのみ
- サンセリフ本文テキスト — すべての本文は等幅必須
- 装飾的イラスト — タイポグラフィと写真のメタファーのみ
- 滑らかなグラデーション背景 — テクスチャとグレインは必須

出力:
1) フィルムストリップ + タイプライターフュージョンレイアウトシステム
2) SVGグレインオーバーレイ + 文字グリッド + セーフライトグローエフェクト
3) レスポンシブブレークポイントと暗室カラートークンを含む完全なHTML/CSS
