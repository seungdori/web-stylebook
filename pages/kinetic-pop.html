BOLD SHAPES, FAST RHYTHM

A style suited for campaign and launch pages. The key is "strong shape contrast" and "short sentence rhythm." When prompting AI, rather than using many colors, it's better to specify 2-3 high-saturation accent points used boldly to push the message forward.

캠페인형/런칭형 페이지에 적합한 스타일입니다. 핵심은 "강한 형태 대비"와 "짧은 문장 리듬"입니다. AI에게 요청할 때는 색을 많이 쓰기보다, 2~3개의 고채도 포인트를 크게 써서 메시지를 밀어붙이도록 지정하는 게 좋습니다.

キャンペーン型・ローンチ型ページに適したスタイルです。ポイントは「強い形状のコントラスト」と「短い文章のリズム」です。AIへのリクエスト時には、多くの色を使うよりも、2〜3個の高彩度アクセントを大きく使ってメッセージを押し出すよう指定するのが効果的です。

AI Request Prompt

Create a landing page for a launch campaign with dynamic, high-contrast pop-art rhythms.

Requirements:
- Large typography (Do Hyeon or bold Korean font) with short sentences arranged rhythmically
- 3 high-saturation accent colors: Yellow (#ffe85d), Blue (#196bff), Pink (#ff4f7b)
- Background: Light paper tone (#f6f3ea) with radial-gradient color bleeds
- Boxes/shapes use thick borders (2-3px solid) + rounded corners (14-36px) for visual guidance
- Place circular blob shapes as background decoration in the hero area
- Motion: Staggered entrance (0.08s intervals, translateY + scale), subtle elastic effect on hover

Responsive:
- Mobile: Cards stack in 1 column, heading clamp(2rem, 7vw, 5.2rem)
- Desktop: max-width 1120px, cards auto-fit minmax(210px, 1fr)

Restrictions:
- No long paragraph text (short copy repetition is the principle)
- No desaturated/neutral tones

Output:
1) Hero / Benefits (3-column bands) / Event / CTA composition
2) Color tokens and component rules (cards/buttons/blobs)
3) HTML/CSS code with responsive support
역동적이고 대비가 강한 팝아트 리듬감으로 런칭 캠페인용 랜딩 페이지를 만들어줘.

필수 조건:
- 대형 타이포(Do Hyeon 또는 굵은 한글 폰트)와 짧은 문장을 중심으로 리듬감 있게 구성
- 고채도 포인트 컬러 3개: 옐로(#ffe85d), 블루(#196bff), 핑크(#ff4f7b)
- 배경: 밝은 종이 톤(#f6f3ea) 위에 radial-gradient로 컬러 번짐 배치
- 박스/도형은 굵은 테두리(2~3px solid) + 둥근 코너(14~36px)로 시선 유도
- 히어로 영역에 원형 블롭(blob) 도형을 배경 장식으로 배치
- 모션: 스태거 진입(0.08s 간격, translateY + scale), hover에서 약한 탄성 효과

반응형:
- 모바일: 카드 1열 스택, 제목 clamp(2rem, 7vw, 5.2rem)
- 데스크톱: max-width 1120px, 카드 auto-fit minmax(210px, 1fr)

금지사항:
- 긴 문단 텍스트 사용 금지(짧은 카피 반복이 원칙)
- 저채도/뉴트럴 톤 사용 금지

출력:
1) 히어로 / 베네핏(3열 밴드) / 이벤트 / CTA 구성안
2) 컬러 토큰과 컴포넌트 규칙(카드/버튼/블롭)
3) 반응형 대응이 포함된 HTML/CSS 코드
ダイナミックでコントラストの強いポップアートのリズム感でローンチキャンペーン用のランディングページを作成してください。

必須条件:
- 大型タイポグラフィ(Do Hyeonまたは太い韓国語フォント)と短い文章を中心にリズミカルに構成
- 高彩度ポイントカラー3色: イエロー(#ffe85d)、ブルー(#196bff)、ピンク(#ff4f7b)
- 背景: 明るい紙トーン(#f6f3ea)の上にradial-gradientでカラーのにじみを配置
- ボックス/図形は太いボーダー(2〜3px solid)+角丸コーナー(14〜36px)で視線誘導
- ヒーローエリアに円形ブロブ図形を背景装飾として配置
- モーション: スタガー進入(0.08s間隔、translateY+scale)、hoverで軽い弾性効果

レスポンシブ:
- モバイル: カード1列スタック、見出しclamp(2rem, 7vw, 5.2rem)
- デスクトップ: max-width 1120px、カードauto-fit minmax(210px, 1fr)

禁止事項:
- 長い段落テキストの使用禁止(短いコピーの反復が原則)
- 低彩度/ニュートラルトーンの使用禁止

出力:
1) ヒーロー / ベネフィット(3列バンド) / イベント / CTA構成案
2) カラートークンとコンポーネントルール(カード/ボタン/ブロブ)
3) レスポンシブ対応を含むHTML/CSSコード
